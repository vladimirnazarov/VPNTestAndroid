**Как запустить проект**

**Клонируйте репозиторий:**
1. _git clone [<repo-url>](https://github.com/vladimirnazarov/VPNTestAndroid)_
2. _Откройте проект в Android Studio._
3. _Синхронизируйте Gradle и дождитесь установки зависимостей._
4. _Запустите проект на эмуляторе или реальном устройстве через MainActivity._
   
**Исправленные проблемы UI**
1. _Упрощён интерфейс, чтобы он был менее нагромождённым._
2. _Скорректировано соотношение главных и второстепенных кнопок._
3. _Убраны лишние элементы, которые не были нужны для тестового задания._

**UX решения**
1. _Проект реализован на Jetpack Compose._
2. _Использован Hilt для внедрения зависимостей и управления ViewModel._
3. _Логика выбора страны и подключения VPN реализована через ViewModel с mutableStateOf._
